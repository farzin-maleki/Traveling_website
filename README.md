# Travel Landing Page

A simple one-page travel website created to practice implementing video background animations using HTML and CSS.

## Technologies Used
- HTML5
- CSS3

## Features
- Fullscreen video background
- Clean one-page layout
- Basic front-end structure

## Purpose of the Project
This project was built as part of my front-end development practice to improve my understanding of video backgrounds and page layout design.
